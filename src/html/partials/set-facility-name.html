Domain ID is the internal code name of the facility. This determines the file directory within your dropbox.

Normally you do this as the first step of site setup, and never change it.

If you change Domain ID later, it will change your dropbox configuration, and may have other impacts. A change to Facility Full Name will not matter, unless there are add-on reporting and data extractions that care.
